## 🏠 Dashboard View

**Purpose:** Get an instant overview of your entire fleet's performance and recent activities.

### Key Metrics Displayed

1. **Total Fleet** - Shows the total number of vehicles in your fleet
2. **Active Now** - Count of vehicles currently in active use or rented
3. **In Maintenance** - Number of vehicles currently undergoing maintenance
4. **Gross Revenue** - Total revenue generated by all vehicles

### Recent Fleet Activity
A real-time feed showing:
- Latest vehicle registrations
- Mileage updates
- Status changes
- Maintenance checks
- Vehicle rentals and returns

Each activity shows:
- Activity type
- Vehicle plate number
- Current status (Completed/Pending/Active)
- Timestamp

### Quick Actions
Two convenient shortcuts on the Dashboard:
- **Manage Registry** - Navigate to your vehicle registry
- **Register Asset** - Quickly add a new vehicle to your fleet

## 📋 Vehicle Registry

**Purpose:** Manage all your vehicles with detailed information and real-time updates.

### Search & Filter
- Use the search box at the top to quickly find vehicles
- Search by vehicle name (e.g., "Mercedes", "Tesla") or license plate
- Results update instantly as you type

### Vehicle Information Displayed

| Column | Description |
|--------|-------------|
| **Vehicle Details** | Make/model and license plate number |
| **Type** | Category: Taxi, Rental, or Logistics (color-coded) |
| **Mileage** | Current odometer reading in kilometers |
| **Fuel** | Fuel level percentage with visual gauge |
| **Status** | Current vehicle status with indicator dot |
| **Actions** | Buttons to update status and mileage |

### Vehicle Statuses
- 🟢 **Available** - Ready for use
- 🔵 **Active** - Currently in use/operational
- 🔵 **Rented** - Currently rented out
- 🔴 **Maintenance** - Under maintenance
- 🔴 **Low Fuel** - Requires fueling
- ⚫ **Inactive** - Not in service

### Managing Individual Vehicles

#### Adding a New Vehicle
1. Click the **"+ New"** button in the Registry section
2. Fill in the following details:
   - **Make & Model** - e.g., "Toyota Corolla"
   - **License Plate** - e.g., "ABC 123"
   - **Type** - Select from: Taxi, Rental, or Logistics
   - **Initial Mileage** - Starting odometer reading
3. Click **"Save Vehicle"** to add to your fleet

#### Updating Vehicle Status
1. Click the **"Status"** button in the Actions column
2. Current status is displayed
3. Select the new status from the dropdown:
   - Available
   - Active
   - Rented
   - Maintenance
   - Low Fuel
   - Inactive
4. Click **"Update Status"** to confirm
5. Activity log updates automatically

#### Updating Vehicle Mileage
1. Click the **"Update km"** button in the Actions column
2. Review the current mileage
3. Enter the new mileage reading (must be higher than current)
4. Click **"Update"** to confirm
5. Change is logged in recent activities

**Note:** Mileage can only be increased, ensuring data integrity.

## 📊 Financial Reports

**Purpose:** Analyze vehicle profitability and operational costs at a glance.

### Report Summary Cards
Displays key financial metrics:
- **Total Revenue** - Combined revenue from all vehicles
- **Total Operational Costs** - Sum of fuel and maintenance costs
- **Net Fleet Profit** - Overall profitability (Revenue - Costs)

### Net Profit per Vehicle Table

Detailed breakdown showing:
| Column | Information |
|--------|-------------|
| **Asset** | Vehicle name and plate |
| **Revenue** | Total earnings from this vehicle |
| **Fuel Cost** | Total fuel expenses |
| **Maint. Cost** | Total maintenance expenses |
| **Net Profit** | Revenue minus all costs |

### Row Totals
Bottom row displays fleet-wide totals for all metrics—quick reference for company-wide performance.

### Export Reports

#### Export as Excel (CSV)
1. Click **"Excel (CSV)"** button
2. File `fleet_report.csv` downloads to your computer
3. Open in Excel, Google Sheets, or any spreadsheet application
4. Use for further analysis, presentations, or record-keeping

#### Export as PDF Report
1. Click **"PDF Report"** button
2. Professional PDF document downloads automatically
3. Includes:
   - Company branding (MAVTAT Motors)
   - Generation timestamp
   - Complete vehicle-by-vehicle financial breakdown
   - Formatted for printing and sharing with stakeholders

## ❓ FAQ & Troubleshooting

### "I can't update mileage to a lower number"
This is by design for data integrity. Mileage readings always move forward. If you need to correct an error, you must set it to a higher value.

### "My data isn't showing in reports"
- Ensure vehicles have revenue values assigned (sampled in demo)
- Check that vehicle status is not set to "Inactive"
- Refresh the page and navigate back to Reports

### "Where does my data save?"
This demo stores all data in your browser's memory. Data is retained during your session. For a production system, data would persist in a cloud database.

### "Can I use this on my phone?"
Yes! The app is fully responsive and works on all modern smartphones. Use the bottom navigation for mobile access.

### "How do I delete a vehicle?"
This demo manages vehicles through status changes (set to "Inactive" to remove from active operations). Production version may include delete functionality.
