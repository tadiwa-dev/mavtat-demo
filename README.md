# MAVTAT Motors Fleet Management Demo

A modern fleet management dashboard built with Tailwind CSS and vanilla JavaScript. This demo showcases a complete vehicle fleet management system with real-time analytics, reporting, and vehicle management capabilities.

## Features

- **Dashboard**: Overview of fleet statistics and recent activities
- **Vehicle Registry**: Manage and search vehicles with detailed specifications
- **Financial Reports**: Comprehensive profit and cost analysis per vehicle
- **Responsive Design**: Fully responsive on mobile, tablet, and desktop
- **PDF & CSV Export**: Generate reports in multiple formats
- **Real-time Updates**: Dynamic UI with smooth transitions and animations

## Quick Start

Simply open `index.html` in your web browser. No build process or dependencies required!

## Technologies Used

- **Tailwind CSS**: Utility-first CSS framework
- **Vanilla JavaScript**: No frameworks, pure JS
- **Material Symbols**: Beautiful icon library
- **jsPDF**: PDF generation for reports

## Deployment

### Deploy to Vercel (Recommended)

1. **Using Vercel CLI**:
   ```bash
   npm install -g vercel
   vercel
   ```

2. **Using Vercel Web Dashboard**:
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import the GitHub repository: `tadiwa-dev/mavtat-demo`
   - Click "Deploy"

3. **Your site will be live at**: `https://mavtat-demo.vercel.app` (or your custom domain)

### Other Deployment Options

- **GitHub Pages**: Push to `gh-pages` branch
- **Netlify**: Connect GitHub repo to Netlify
- **Firebase Hosting**: Deploy using Firebase CLI
